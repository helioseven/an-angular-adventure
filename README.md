# Contribution Guide
## Quickstart ##
+ Install Unity Hub
    * Unity 6.4 - Current build is using `6000.4.0f1`
+ Install Git
+ Clone this Repo
    * Main branch is `origin/master`
+ Install Steamworks.NET
    * [Read installation instructions.](https://steamworks.github.io/installation/)
    * Current build is using `2025.162.1`
